# 웹페이지 URL 요약 스킬

사용자가 URL을 제공하면, 해당 웹페이지의 내용을 가져와 한국어로 명확하게 요약 정리한다.

## 동작 순서

1. **URL 수집**: 사용자가 제공한 URL을 확인한다.
2. **페이지 내용 가져오기**: `WebFetch` 도구를 사용해 페이지 본문을 가져온다.
3. **콘텐츠 유형 파악**: 페이지 내용을 보고 어떤 종류인지 판단한다 (뉴스, 기술 문서, 쇼핑, YouTube 등).
4. **적응형 요약 생성**: 콘텐츠 유형에 맞는 포맷으로 한국어 요약을 작성한다.
5. **결과 전달**: 아래 출력 형식에 따라 깔끔하게 정리해서 사용자에게 제공한다.

## 콘텐츠 유형별 요약 형식

### 뉴스 / 블로그 / 일반 기사

```
## 📰 [페이지 제목]
**출처**: [도메인 또는 사이트명]

### 핵심 요약 (3~5줄)
[기사의 핵심 내용을 3~5문장으로 간결하게 요약]

### 주요 포인트
• [핵심 포인트 1]
• [핵심 포인트 2]
• [핵심 포인트 3]
(필요시 더 추가)
```

### 기술 문서 / 개발 가이드

```
## 📄 [문서 제목]
**출처**: [도메인 또는 사이트명]

### 문서 목적
[이 문서가 무엇을 다루는지 1~2문장]

### 주요 내용
• [핵심 개념 또는 기능 1]
• [핵심 개념 또는 기능 2]
• [핵심 개념 또는 기능 3]

### 결론 / 핵심 정리
[문서에서 가장 중요한 결론이나 사용법 요약]
```

### 쇼핑 상품 페이지

```
## 🛍️ [상품명]
**출처**: [쇼핑몰명]

### 핵심 정보
• **가격**: [가격 정보, 없으면 "확인 불가"]
• **카테고리**: [상품 종류]

### 주요 특징
• [특징 1]
• [특징 2]
• [특징 3]

### 한줄 요약
[이 상품이 어떤 사람에게 적합한지 한 문장으로]
```

### YouTube 영상 페이지

```
## 🎬 [영상 제목]
**채널**: [채널명]
**출처**: YouTube

### 영상 소개 요약
[영상 설명(description) 텍스트를 기반으로 내용 요약]

### 주요 포인트
• [설명에서 파악한 핵심 내용 1]
• [설명에서 파악한 핵심 내용 2]

> ⚠️ 참고: 영상 자체는 재생할 수 없으므로, 페이지의 제목과 설명 텍스트를 기반으로 요약합니다.
```

### 기타 / 일반 웹페이지

위 유형에 해당하지 않는 경우, 뉴스/블로그 형식을 기본으로 사용하되 내용에 맞게 자유롭게 조정한다.

## 예외 처리 — 솔직하게 안내하기

분석이 불가능한 경우, 이유를 명확하게 설명하고 사용자에게 솔직하게 전달한다. 이유를 숨기거나 애매하게 처리하지 않는다.

| 상황 | 안내 방식 |
|------|-----------|
| 로그인이 필요한 페이지 | "이 페이지는 로그인 후에만 접근 가능합니다. 로그인 없이는 내용을 가져올 수 없어 요약이 불가합니다." |
| 접근이 차단된 페이지 (403, paywall 등) | "이 페이지는 접근이 제한되어 있습니다. (예: 유료 구독 필요, 특정 지역 차단 등) 내용을 가져올 수 없습니다." |
| URL이 잘못되었거나 페이지가 없음 (404) | "해당 URL의 페이지를 찾을 수 없습니다. URL을 다시 확인해 주세요." |
| 이미지만 있는 페이지 | "이 페이지는 텍스트 없이 이미지로만 구성되어 있어 내용을 읽을 수 없습니다. 요약이 불가합니다." |
| JavaScript로만 렌더링되는 페이지 | "이 페이지는 JavaScript를 실행해야 내용이 표시되는 구조입니다. 현재 환경에서는 내용을 가져오기 어렵습니다." |
| 네트워크 오류 / 타임아웃 | "페이지를 불러오는 중 오류가 발생했습니다. 잠시 후 다시 시도하거나, URL이 올바른지 확인해 주세요." |

## 출력 원칙

- **모든 요약은 반드시 한국어로** 작성한다. 원문이 영어이거나 다른 언어여도 한국어로 번역해서 요약한다.
- 요약은 **사실에 충실하게** 작성한다. 페이지에 없는 내용을 추측하거나 지어내지 않는다.
- 너무 길지 않게, 핵심만 담아 **읽기 쉽게** 작성한다.
- 전문 용어가 포함된 경우, 필요하다면 짧은 설명을 덧붙인다.
