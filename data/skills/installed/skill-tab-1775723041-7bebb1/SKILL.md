---

## 동작 방식

사용자가 제공한 텍스트의 글자수를 세어 결과를 출력한다.

## 세는 규칙

- 모든 문자(한글, 영문, 숫자, 특수문자)를 각 1자로 센다
- 공백(띄어쓰기)도 1자로 센다
- 줄바꿈(엔터)도 1자로 센다
- 추가 설명이나 부가 정보는 출력하지 않는다

## 출력 형식

```
총 N자입니다
```

N은 실제 글자수로 대체한다. 이 한 줄만 출력한다.

## 빈 입력 처리

텍스트가 비어 있거나 아무것도 없으면 다음과 같이 응답한다:

```
입력된 텍스트가 없습니다.
```

## 예시

입력: `안녕 세상`
출력: `총 6자입니다`

입력: `안녕 세상\n끝` (줄바꿈 포함)
출력: `총 8자입니다`

입력: (없음)
출력: `입력된 텍스트가 없습니다.`
